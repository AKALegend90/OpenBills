# OpenBills v1

First public release of OpenBills, a free, open-source, privacy-first bill tracker.

## Highlights

- Manual bill tracking with paid/unpaid status
- Manual income tracking with monthly estimates
- Dashboard totals for income, bills, money left, due today, due soon, and overdue bills
- Calendar view with all-month jump picker
- Monthly reset and payment history
- Dark mode
- CSV and JSON import/export
- Optional Windows startup, start minimized, tray menu, and local system notifications
- Close-to-tray behavior with right-click Quit
- Local-only storage with no account, bank connection, ads, analytics, or cloud sync

## Windows Download

Download `OpenBills.exe` from this release.

Because this test build is unsigned, Windows may show SmartScreen. Click **More info** then **Run anyway** if prompted.
